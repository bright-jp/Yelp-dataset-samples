# Yelp-dataset-samples

<h2>1001件のYelpビジネスレビューのサンプルデータセット</h2>

![Yelp dataset header](https://github.com/luminati-io/Yelp-dataset-samples/blob/main/yelp-datasets.png)

1000件以上のYelpビジネスレビューのYelpデータセットサンプルです。データセットは <b>Bright Data API</b> を使用して抽出されています。

<h2>データセットに含まれるデータポイントの一部：</h2>

* ```business_id```: ビジネスの一意の識別子です
* ```Review_author```: レビューを書いた著者またはユーザーです
* ```Rating```: レビュアーが付けた評価です
* ```Date```: レビューが投稿された日付です
* ```Content```: レビューのテキスト内容です
* ```Review_image```: レビューに関連付けられた画像です
* ```Reactions```: レビューに対して他のユーザーから受け取ったリアクションです
* ```Replies```: レビューへの返信または応答です
* ```review_order```: 同一ビジネスにおける他のレビューの中でのレビューの順序または連番です
* ```Elite_status```: レビューの著者がエリートステータスであるかどうかを示します

ほかにも多数あります。

これは「Yelp business reviews」データセットから派生したサンプルサブセットであり、このデータセットには <b>2億件以上のレコード</b> が含まれます。

利用可能なデータセットのファイル形式：<b>JSON, NDJSON, JSON Lines, CSV, または Parquet。必要に応じて、ファイルを .gz に圧縮できます</b>。

データセット配信タイプのオプション：<b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>。

更新頻度：<b>Once, Daily, Weekly, Monthly, Quarterly, または Custom basis</b>。

抽出したデータポイントに追加できるデータエンリッチメント：<b>Based on request.</b>

<b>[Yelpのフルデータセットを入手する](https://brightdata.jp/products/datasets/yelp)</b>。

<h2>Yelpデータセットのユースケースは何ですか？</h2>

<h3>1. 消費者センチメント分析</h3>
自社ビジネス、競合、業界トレンドに関するレビューやコメントを分析することで、顧客フィードバックの洞察を得られます。この分析を活用して、商品提供の強化、サービス改善、顧客体験のパーソナライズを行えます。これにより、顧客ニーズに先回りして対応し、競争力を維持できます。

<h3>2. 市場調査と成長機会</h3>
ビジネス所在地、評価、顧客レビューの変化をモニタリングし、新たな市場トレンドや拡大の可能性がある領域を特定できます。業界の変化を最新の状態で把握することで、成長機会を能動的に見つけ、データに基づく意思決定によって市場でのポジションを強化できます。

<h3>3. B2Bデータを用いた競合分析</h3>
レストラン、ホームサービス、自動車サービスなど、競合ビジネスに関するデータを収集し、営業時間、所在地、顧客レビュー、サービス内容を分析できます。このデータを活用することで、競合の強みと弱みをより深く理解でき、自社の戦略を洗練し、ビジネスのポジショニングを改善できます。
